# FairWrite

AI-Powered Bias Detection and Inclusive Writing Platform

## Features

- Bias Detection using AI
- Inclusive Text Rewriting
- PDF Upload & Analysis
- Analysis History Tracking
- User Authentication
- PDF Report Export

Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- FastAPI
- Python
- MySQL

AI:
- Google Gemini API
